# Lynx BI - Brand Guidelines

## Brand Name
**Lynx BI**

### Pronunciation
"Links Bee Eye"

### Meaning
- **Lynx**: Symbolizes sharp vision, agility, and precision - qualities essential for business intelligence
- **BI**: Business Intelligence

## Visual Identity

### Logo Text
```
Lynx BI
MIPS Analytics
```

### Typography
- Primary: Space Grotesk (headings)
- Secondary: Inter (body text)
- Logo: Bold, gradient effect

### Color Palette

**Primary Colors:**
- Accent Indigo: `#6366f1`
- Accent Purple: `#a855f7`
- Accent Pink: `#ec4899`

**Secondary Colors:**
- Neon Cyan: `#06b6d4`
- Neon Green: `#10b981`
- Neon Blue: `#3b82f6`

**Neutral Colors:**
- Dark Background: `#0a0a0f`
- Surface: `#1a1a2e`
- Text Primary: `#ffffff`
- Text Secondary: `#94a3b8`

### Gradients

**Primary Gradient:**
```css
background: linear-gradient(to right, #6366f1, #a855f7);
```

**Glow Effect:**
```css
text-shadow: 0 0 30px rgba(99, 102, 241, 0.5);
```

## Tone of Voice

### Characteristics
- **Professional**: Enterprise-grade solution
- **Innovative**: Cutting-edge technology
- **Confident**: "Reimagined", "Unprecedented"
- **Technical**: MIPS, Assembly, Analytics
- **Modern**: 3D, Real-time, Advanced

### Writing Style
- Clear and concise
- Technical but accessible
- Action-oriented
- Future-focused

## Usage Examples

### Correct ✅
- "Lynx BI - Business Intelligence Platform"
- "Powered by Lynx BI"
- "Welcome to Lynx BI"
- "Lynx BI combines..."

### Incorrect ❌
- "LynxBI" (no space)
- "Lynx-BI" (hyphenated)
- "lynx bi" (lowercase)
- "LYNX BI" (all caps, except in logos)

## Competitive Positioning

### Differentiators
1. **MIPS Assembly Engine**: Unique low-level performance
2. **3D Visualizations**: Modern, immersive experience
3. **Open Architecture**: Transparent and extensible
4. **Developer-Friendly**: Assembly code visible and modifiable

### Comparison Points
- vs Power BI: More performant, open-source
- vs Tableau: More technical, customizable
- vs Looker: More innovative, 3D visuals
